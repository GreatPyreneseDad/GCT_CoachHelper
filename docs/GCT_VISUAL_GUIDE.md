## How Coherence is Calculated

```
Step 1: Measure Each Dimension (0 to 1 scale)
┌────────────────────────────────────────────┐
│  Ψ = 0.7  │  ρ = 0.6  │  q = 0.5  │  f = 0.8  │
└────────────────────────────────────────────┘

Step 2: Apply the Formula
┌─────────────────────────────────────────────────────┐
│                                                     │
│  Coherence = Ψ + (ρ × Ψ) + q_optimal + (f × Ψ)    │
│                                                     │
│  Coherence = 0.7 + (0.6 × 0.7) + 0.45 + (0.8 × 0.7)│
│                                                     │
│  Coherence = 0.7 + 0.42 + 0.45 + 0.56              │
│                                                     │
│  Coherence = 2.13                                   │
│                                                     │
└─────────────────────────────────────────────────────┘

Step 3: Normalize to Percentage
┌─────────────────────────────────────────────────────┐
│  Maximum Possible = 4.0                             │
│  Client Score = 2.13                                │
│  Percentage = (2.13 / 4.0) × 100 = 53.25%          │
└─────────────────────────────────────────────────────┘
```
