## Session Flow Template

1. **Check-in** (5 min)
   - Quick assessment of 4 dimensions
   - Identify biggest challenge

2. **Focus** (30 min)
   - Work on lowest dimension
   - Use targeted intervention

3. **Integration** (10 min)
   - Connect to other dimensions
   - Set weekly practice

4. **Commitment** (5 min)
   - One specific action
   - Schedule follow-up
